# Unique Word Counter
Write a non-trivial program in assembly language

## Summary
Prompt user for a text file input.
Return the most commonly used word in the text file to standard output.

## Run and Debug on Docker
#### Build Image 
```
make build-image
```

#### Compile and Link
```
make build
```

#### Run 
```
make run
```

#### Debug
```
make debug
```
